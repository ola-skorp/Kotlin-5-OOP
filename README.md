# Kotlin-5

Есть животные (кошка, попугай, крокодил), 
у которых есть уровень красоты и метод, чтобы увеличить его.
Есть части тела животного, которые можно обработать на груминге.
Их можно мыть/чистить, расчёсывать и стричь.
Груминг повышает уровень красоты животных.
Крокодил кусается каждый раз при попытке сделать его более красивым.
Есть владельцы животных, которые могут иметь много животных.
У владельцев животных есть список частей тела животных, 
которые владельцы хотят обработать на груминге.
Владельцем животных можно стать, купив животных в магазине или подобрав их на улице.
В зоомагазине владелец может выбрать животных.
На улице владелец всегда получает не то животное, которое выбрал.
Есть груминг салон, который может обслуживать владельцев животных и засудить владельца, 
если животное сделало кусь в процессе груминга.

В методе main генерируем рандомных владельцев, 
создаём единственный существующий груминг салон и обслуживаем всех сгенерированных владельцев.
Процесс обслуживания выводим в консоль
